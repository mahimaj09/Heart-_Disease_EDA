# ❤️ Heart Disease Exploratory Data Analysis (EDA)

> An Exploratory Data Analysis (EDA) project using Python to uncover patterns, trends, and key health indicators associated with heart disease through data visualization and statistical analysis.

---

## 📌 Project Overview

This project analyzes a Heart Disease dataset to understand the factors influencing heart disease. Using Python and various data visualization techniques, the dataset was explored to identify meaningful patterns, correlations, and insights that can support healthcare decision-making and serve as a foundation for predictive modeling.

---

## 🎯 Objectives

- Explore the dataset and understand its structure
- Perform data quality checks
- Analyze patient demographics and medical attributes
- Visualize relationships between important features
- Identify factors associated with heart disease
- Generate actionable insights from the data

---

## 🛠️ Tech Stack

- **Python**
- **Pandas**
- **NumPy**
- **Matplotlib**
- **Seaborn**
- **Plotly**

---

## 📊 Exploratory Data Analysis

The following visualizations were created during the analysis:

- ✅ Heart Disease Distribution
- ✅ Gender Distribution
- ✅ Age Distribution
- ✅ Age vs Maximum Heart Rate (Scatter Plot)
- ✅ Age Distribution by Heart Disease (Box Plot)
- ✅ Chest Pain Type Distribution
- ✅ ST Segment Slope Distribution
- ✅ Fasting Blood Sugar vs Heart Disease
- ✅ Age vs Cholesterol (Regression Plot)
- ✅ Correlation Heatmap
- ✅ Pairplot of Important Features

---

## 🔍 Key Insights

- Patients with heart disease are generally older.
- Higher **Oldpeak** values are strongly associated with heart disease.
- Lower **Maximum Heart Rate (MaxHR)** is commonly observed among heart disease patients.
- **Chest Pain Type** appears to be one of the strongest indicators of heart disease.
- Cholesterol alone does not show a strong relationship with heart disease.
- Fasting Blood Sugar has only a weak association with the target variable.
- Correlation analysis suggests that multiple features together provide better predictive information than any single feature.

---

## 📌 Future Improvements

- Perform feature engineering
- Build machine learning models for heart disease prediction
- Compare classification algorithms
- Create an interactive Power BI dashboard
- Deploy the project using Streamlit
